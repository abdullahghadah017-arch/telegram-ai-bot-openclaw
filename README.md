# Telegram AI Bot using OpenClaw

An AI-powered Telegram bot integrated with OpenClaw that enables interactive communication using AI models.

## 💡 Description
This project connects a Telegram bot with OpenClaw, allowing users to send messages and receive AI-generated responses in real-time.

## ⚙️ Features
- Telegram Bot Integration
- OpenClaw Gateway Connection
- AI-powered responses
- Real-time communication

## 🛠️ Technologies Used
- OpenClaw
- Telegram Bot API
- Local AI Models (Ollama)

## ▶️ How it Works
1. User sends a message on Telegram
2. Message is processed via OpenClaw
3. AI model generates a response
4. Response is sent back to the user
